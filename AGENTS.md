# Agents

## Agent skills

### Issue tracker

Issue 跟踪使用本地 Markdown 文件推进，不使用 GitHub Issues 或 `gh` CLI。See `docs/agents/issue-tracker.md`.

### Triage labels

本项目不使用远程 issue 标签。任务文件内使用 `type` 和 `status` 字段表达 AFK/HITL 与推进状态。See `docs/agents/triage-labels.md`.

### Domain docs

单上下文布局，`CONTEXT.md` + `docs/adr/` 位于仓库根目录。See `docs/agents/domain.md`.
