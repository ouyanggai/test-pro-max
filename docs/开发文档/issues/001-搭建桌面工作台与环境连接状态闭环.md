## What to build

搭建 Python 桌面工具的最小工作台，按 UI 设计规范形成三栏布局、顶部工具栏和底部状态栏，并打通环境配置读取。用户打开工具后能看到当前环境、API 网关、连接状态和密钥缺失提示。

## Acceptance criteria

- [x] 工具启动后展示左侧导航、中央主操作区、右侧详情面板、顶部工具栏、底部状态栏。
- [x] 能读取本地非密钥环境配置，并展示 API 网关和环境名称。
- [x] 密钥缺失时给出明确提示，但不输出密钥内容。
- [x] 环境连接检查成功、失败、未配置三种状态能在底部状态栏展示。
- [x] 相关测试保存到 `test` 目录并按配置/环境分类。

## Notes
