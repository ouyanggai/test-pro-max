## 状态说明

| 字段 | 取值 | 说明 |

## 推进规则

- 新任务从 `todo` 开始。
- 开始实现前将对应任务文件的 `status` 改为 `doing`。
- 需要人工确认时改为 `blocked`，并在 `Notes` 写明阻塞点。
- 完成并验收后改为 `done`。
- 不重排已有编号；新增任务从 015 开始。
