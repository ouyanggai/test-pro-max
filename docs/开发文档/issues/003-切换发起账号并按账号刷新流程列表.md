## What to build

用户可以搜索并选择其他发起账号。工具使用该账号获取会话，并刷新为该账号可发起的流程列表。

## Acceptance criteria

- [ ] 支持按姓名搜索用户。
- [ ] 支持按部门和岗位筛选用户。
- [ ] 选择用户后刷新发起账号显示。
- [ ] 流程列表按新账号权限重新查询。
- [ ] 用户查询结果展示姓名、账号、部门、岗位和启用状态。

## Notes

用户查询由 `UserDirectoryService` 封装。无权限账号应返回空列表或明确错误，不影响默认账号配置。
