## What to build

用户打开工具后，系统默认以“欧阳改”作为超级发起账号获取会话，并查询该账号可发起流程。流程列表在中央主操作区展示，登录和流程查询过程写入运行日志。

## Acceptance criteria

- [ ] 默认账号显示为“欧阳改”。
- [ ] 默认账号登录成功后缓存 SID，不在 UI 或日志中明文显示 SID。
- [ ] 可发起流程列表展示流程名称、分类、流程标识和来源账号。
- [ ] 登录失败、无权限、网关不可用时展示明确错误。
- [ ] 登录和流程查询行为有可测试的服务接口。

## Notes

流程查询应基于真实 API 网关。不得把密码、验证码或 SID 写入仓库。
