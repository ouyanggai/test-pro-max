# 接口编排自动化测试工具 — 实施计划索引

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement each task file individually.

**Goal:** 从零构建 PySide6 桌面工具，实现"选账号→选流程→填表单→配节点→运行→看报告"六步向导，涵盖 10+ 核心服务模块。

**Architecture:** UI 层（PySide6）→ 业务编排层 → 核心服务层 → RSH Cloud 后端（httpx.AsyncClient）→ SQLite本地存储。qasync 融合 Qt 事件循环和 asyncio。

**Tech Stack:** PySide6==6.11.0, qt-material==2.17, pytest==9.0.2, pytest-asyncio==0.25.2, pytest-qt==4.6.0, qasync==0.26.0, httpx==0.28.1, aiosqlite==0.21.0
